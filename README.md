# 손길도우미 확장 프로그램 개발 가이드

손길 도우미 확장프로그램 레포지토리입니다.

---

## 1. 사전 준비

- Node.js ≥ 16, pnpm ≥ 7
- Chrome(MV3) 지원 브라우저

## 2. 의존성 설치

```
pnpm install
```
